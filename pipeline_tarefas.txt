PIPELINE DO GERENCIADOR DE TAREFAS

1. criação/verificação do arquivo de tarefas*
1.1 verificar se existe o arquivo csv
1.2 se não existir, criar o arquivo

2. add tarefas:
*
2.1 - se existir, add tarefa na ultima linha do arquivo

3. listar tarefas:
*
3.1 ler o arquivo csv
3.2 listar todas as linhas (tarefas) = guardar numa variável (Série, lista etc)

4. deletar tarefas:
*
4.1 ler o arquivo csv
4.2 listar todas as linhas (tarefas) = guardar numa variável (Série, lista etc)
4.3 Receber o input do index da tarefa a ser deletada
4.4 Deletar o objeto de acordo com o index
4.5 Sobrescrever csv
